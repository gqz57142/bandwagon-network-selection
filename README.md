## 先搞清楚一件事：线路决定快慢，不是机房位置

很多人挑机房只看物理距离，觉得香港最近所以最快，日本其次，美国太远……这个逻辑对了一半，但没说完。

实际上影响速度的是**线路等级**，也就是数据包从你的运营商出国、在海外传输、再回来走的是哪条路。简单说，搬瓦工涉及这几种主要线路：

- **CN2 GIA**：电信最高等级国际专线，全程走 AS4809 节点，不绕普通骨干网。延迟低、晚高峰稳定是最大特点。
- **CN2 GT**：比 CN2 GIA 低一档，普通时段够用，晚高峰偶尔会炸。
- **CUP（AS10099）**：联通面向海外的精品线路，进国内后转接 AS9929，联通用户专属利好。
- **CMIN2**：移动版的精品国际骨干网，移动用户走这条线路延迟低、丢包少。
- **日本软银（BBTEC）**：联通用户连日本大阪软银机房体验极好，延迟低，晚高峰也不怎么炸。

说实话，以前搬瓦工的 DC6、DC9 只是电信 CN2 GIA，联通和移动用户绕着走。但现在这两个机房已经同时接入 CUP 和 CMIN2，等于三网都有了精品线路——这是近两年升级最大的变化。

## 一个容易忽略的操作：自己测一下再买

搬瓦工每个机房都有测试 IP 和测速文件，购买之前完全可以先用 ping 命令测一下延迟，不用瞎猜。

简单操作方法：

1. 打开 CMD（Windows）或终端（Mac/Linux）
2. 输入 `ping [机房测试 IP]` 回车
3. 看平均延迟和丢包率

比如洛杉矶 DC6 的测试 IP 是 `104.243.29.224`，香港 HKHK_8 是 `93.179.124.1`，日本大阪软银是 `199.180.114.1`。测一遍比看几十篇文章都直接。

晚高峰（19:00-23:00）的数据更有参考价值，这时候普通线路和精品线路的差距才能真正体现出来。
